Este projeto tem como objetivo o desenvolvimento de uma página de captura (landing page) utilizando apenas HTML e CSS, com foco no aprendizado prático de estruturação e estilização de interfaces web. A proposta é aplicar conceitos fundamentais como semântica HTML, organização de layout, uso de classes e responsividade básica.

Durante a construção, são explorados elementos essenciais de uma página de conversão, como títulos atrativos, formulários de captura de dados e seções visuais bem definidas. No CSS, o foco está na criação de um design limpo e funcional, trabalhando cores, tipografia e espaçamento para melhorar a experiência do usuário.

Este projeto serve como base para iniciantes que desejam entender como páginas simples são estruturadas e como o front-end pode ser utilizado para gerar resultados práticos, como captação de leads.
